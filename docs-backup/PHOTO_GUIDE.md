## 촬영 팁

### Before 사진 (일부러 지저분하게)
- **핵심:** 실제 셀러가 찍는 수준의 사진. 너무 깔끔하면 Before/After 차이가 안 남
- 자연광 사용 (창가, 낮 시간)
- 배경에 생활 잡동사니가 보이게 (리모컨, 펜, 책 등)
- 살짝 비뚤게 찍어도 됨 — "폰으로 대충 찍은" 느낌
- 그림자가 여러 방향으로 나와도 OK
- **하지 말 것:** 일부러 흔들거나 초점을 흐리지는 말기 — 상품은 선명해야 AI가 잘 처리함

### After 사진 (BgSwap으로 처리)
- 실제로 BgSwap에 업로드해서 나온 결과물 사용
- 각 세트마다 15배경 중 대표 1개를 메인으로, 나머지 4개는 보조
- After 이미지 크롭/리사이즈 하지 말 것 — 원본 그대로 사용해야 신뢰도 높음

### 카메라
- 스마트폰 OK (실제 서비스가 "폰 사진도 된다"고 홍보하므로 오히려 일관성 있음)
- 세로 또는 정사각형 비율
- 해상도: 최소 1000×1000px 이상

## 세트별 촬영 시나리오

### 세트 1: 신발
- **준비물:** 운동화/스니커즈 1켤레
- **Before 세팅:** 현관/거실 바닥, 주변에 신발장/우산/열쇠 등
- **포인트:** 바닥 질감(타일, 나무)이 보여야 Before 느낌 강함
- **After 대표:** White (#FFFFFF) — "Amazon Ready" 태그

### 세트 2: 가방
- **준비물:** 작은 가방, 파우치, 또는 지갑
- **Before 세팅:** 침대/소파 위, 구겨진 이불이나 쿠션 옆
- **포인트:** 패브릭 배경의 주름/무늬가 보여야 Before 느낌
- **After 대표:** Light Gray (#F5F5F5) — "Etsy Ready" 태그

### 세트 3: 화장품
- **준비물:** 스킨케어 병, 크림, 또는 작은 화장품
- **Before 세팅:** 욕실 선반 또는 세면대 위, 치약/칫솔/수건 등 함께
- **포인트:** 생활감 있는 욕실 배경
- **After 대표:** Warm Cream (#FAF5EE) — "Shopify Ready" 태그

### 세트 4: 전자기기
- **준비물:** 이어폰, 충전기, 또는 작은 가제트
- **Before 세팅:** 책상 위, 키보드/마우스/커피잔/케이블 등 함께
- **포인트:** 어수선한 데스크 환경
- **After 대표:** Dark (#1E1E23) — "Premium" 태그

## 촬영 후 작업 흐름

1. Before 사진 4장 확보
2. 각 사진을 bgswap.io에 업로드 → 15배경 결과 다운로드
3. Before 1장 + After 대표 1장 = 1세트 (랜딩 메인용)
4. After 나머지 4장 = 보조 (15배경 미리보기 섹션용)

## 랜딩 페이지 적용 위치

| 섹션 | 현재 | 변경 후 |
|------|------|---------|
| "See the Difference" (Before/After) | CSS 시뮬레이션 + 이모지 | 실사 Before/After 2세트 (신발 + 화장품 권장) |
| "15 Backgrounds, Every Photo" | 이모지 📦 스와치 | 실사 상품 1개의 15배경 결과물 |
| Product Hunt 스크린샷 3 | 미정 | 15배경 나란히 배치 이미지 |

## 이미지 최적화 (코드 적용 시)

- Next.js `<Image>` 컴포넌트 사용
- WebP 포맷, srcset 자동 생성
- alt 텍스트 예시:
  - Before: "Sneaker on messy floor before BgSwap"
  - After: "Same sneaker on clean white background after BgSwap"
- lazy loading (첫 번째 Before/After만 eager)
- 파일 저장 경로: `public/samples/`
  - `before-shoes.jpg`, `after-shoes-white.jpg`, `after-shoes-gray.jpg` ...
