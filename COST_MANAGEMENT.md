## 1. 서비스별 비용 상세

### 1-1. Replicate (AI 배경 제거)

**모델:** lucataco/remove-bg (BiRefNet) — 커뮤니티 모델, T4 GPU
**과금:** 실행 시간 기반
**공식 단가:** ~$0.0004/건 (warm, ~2초)
**실효 단가 (cold start 포함):** ~$0.0008~0.0012/건 (cold start 30% 비율, 5~10초)

| 시나리오 | 호출 수 | 비용 (warm) | 비용 (cold 포함) |
|---------|--------|------------|-----------------|
| 무료 체험 1건 | 1회 | $0.0004 | $0.001 |
| Starter 결제 (10상품) | 10회 | $0.004 | $0.01 |
| Pro 결제 (100상품) | 100회 | $0.04 | $0.10 |
| 하루 무료 체험 50건 | 50회 | $0.02 | $0.05 |
| 하루 Pro 5건 | 500회 | $0.20 | $0.50 |
| **월 최대 예상** | **5,000회** | **$2.00** | **$5.00** |

> 참고: BUSINESS.md의 "$0.002/장"은 이전 Bria 모델 기준. BiRefNet 교체 후 단가 하락.
> 실제 단가는 Replicate 청구 내역에서 확인 필요.

**크레딧 잔액 확인:** Replicate 대시보드 Billing 탭
**권장:** 최소 $10 크레딧 유지. 월 $5 자동 충전 설정

**리스크:**

| 리스크 | 심각도 | 대응 |
|--------|--------|------|
| 크레딧 $5 미만 → rate limit 분당 6건 | 높음 | $10+ 충전 유지 |
| 크레딧 소진 → 서비스 완전 중단 | 치명 | 자동 충전 설정 |
| 모델 owner가 비활성화/삭제 | 중간 | 대체 모델 후보: `cjwbw/rembg`, Replicate 공식 BiRefNet |
| Replicate 과금 모델 변경 (시간→건당) | 낮음 | 과금 변경 공지 모니터링 |
| cold start로 60초 타임아웃 초과 | 중간 | replicate.ts에 45초 타임아웃 + 3회 재시도 구현됨 |

**자체 호스팅 대안 (모델 삭제/가격 폭등 시):**
- RunPod/Banana: GPU 서버리스, BiRefNet Docker 직접 배포
- 예상 비용: $0.0002~0.0005/건 (더 저렴)
- 개발 비용: 1~2일 작업 (Docker + API 래퍼)

**비용 최적화:**
- 무료 체험 남용 방지: IP 기반 제한 + 일일 한도 100건 (구현됨)
- 불필요한 재시도 방지: failed → retry 최대 3회 (구현됨)
- 무료 체험은 이메일 발송 안 함 → Resend 비용 무관

**현재 사용량 예측:**

| 시나리오 | 월 저장량 | 월 쓰기 | 월 읽기 | 예상 비용 |

## 4. Polar vs Stripe 수수료 비교

현재 Polar (4% + $0.40)를 쓰고 있는데, 결제량이 늘면 Stripe (2.9% + $0.30)가 유리해짐.

| 월 결제 | Polar 수수료 | Stripe 수수료 | 차이 |
|--------|-------------|-------------|------|
| Starter 10건 | $6.00 | $4.45 | Stripe $1.55 절약 |
| Pro 10건 | $15.60 | $11.41 | Stripe $4.19 절약 |
| Pro 50건 | $78.00 | $57.05 | Stripe $20.95 절약 |
| Pro 100건 | $156.00 | $114.10 | Stripe $41.90 절약 |

**전환 시점:** 월 Pro 30건+ 이면 Stripe 전환 비용(개발 1~2일) 대비 효과 있음.
**당분간:** Polar 유지 (결제 연동 이미 구현, 전환 비용 > 절약액)

## 6. 환경변수 관리
